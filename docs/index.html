Core Areas

The main study lanes

Foundations

Dirac delta functions, Poisson’s equation, and Laplace’s equation establish the electrostatic language.

Boundary Methods

Green’s functions and the method of images turn boundary-value ideas into concrete solution tools.

Structure of Fields

Multipole expansion and dielectrics explain how geometry and material response shape the field.

What it adds

The viewer turns a static electrostatics solve into something spatial: you can rotate the periodic 3D solution and inspect where charge density and potential structure live in the same volume.

Charge Hunt

Drag a detector over a hidden line-charge profile and use the response of a narrow delta-sequence to infer where the source is strongest.

Open the game Original Python version

Why it fits the project

The game turns the distribution idea into something physical: the detector response is strongest when the sampling kernel is centered on a localized source.
